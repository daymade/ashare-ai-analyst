# 隔夜跨市场传导分析 (Overnight Cross-Market Transmission Analysis)

## 分析框架

每个交易日开盘前，基于隔夜全球市场表现推演对 A 股的影响。

## 传导链路

### 美股 → A 股

| 美股表现 | 传导路径 | A 股影响板块 | 敏感度 |
|---------|---------|-------------|--------|
| 纳斯达克涨 | 科技情绪传导 | 半导体/消费电子/AI算力 | 高(0.75-0.85) |
| 标普500涨 | 风险偏好提升 | 消费/金融/出口 | 中(0.45-0.55) |
| 道琼斯涨 | 工业/周期情绪 | 机械/建材/基建 | 中(0.40-0.50) |
| 美股暴跌(>2%) | 恐慌传导 | 全市场低开 | 高(0.60-0.80) |

### 大宗商品 → A 股

| 商品表现 | 传导路径 | A 股影响 |
|---------|---------|---------|
| 原油涨 | 成本上升 | 石油/油运↑，航空/化工↓ |
| 原油跌 | 成本下降 | 石油/油运↓，航空/消费↑ |
| 黄金涨 | 避险情绪 | 贵金属↑，科技成长↓ |
| 铜涨 | 经济预期乐观 | 有色金属/基建↑ |

### 汇率 → A 股

| 汇率变动 | 传导路径 | A 股影响 |
|---------|---------|---------|
| 美元走强(人民币贬值) | 出口竞争力↑ | 出口企业↑，进口企业↓ |
| 美元走弱(人民币升值) | 外资流入预期 | 金融/消费↑，出口↓ |

## 关键判断规则

### 美股异动分类

1. **科技领涨型**（纳指涨幅 > 道指）
   - 原因通常：AI/芯片/科技财报超预期
   - A 股看：半导体 + 消费电子 + AI 算力
   - 典型传导：英伟达涨→A股光模块/算力涨

2. **防御领涨型**（道指涨 > 纳指，黄金涨）
   - 原因通常：地缘冲突/衰退担忧
   - A 股看：贵金属 + 军工 + 公用事业
   - 科技板块可能反而承压

3. **全面大跌型**（三大指数均跌>1%）
   - A 股大概率低开
   - 策略：不追高，等回落，现金为王
   - 关注超跌反弹机会（跌幅最大板块次日反弹概率大）

4. **分化型**（指数涨跌不一致）
   - 需要细看驱动因素（政策/财报/事件）
   - 按具体驱动判断传导

### 原油特殊规则

- 油价涨+美股涨 = 通胀担忧降低（正常）
- 油价涨+美股跌 = 滞胀恐慌（危险信号）
- 油价跌+美股涨 = 最佳组合（成本降低+情绪好）
- 油价跌+美股跌 = 衰退恐慌（危险）

### 注意事项

1. **不要过度外推**: 美股涨3%不等于A股也涨3%，A股有独立逻辑
2. **关注反向信号**: 如果美股大涨但A股高开低走，说明A股有自己的利空
3. **时间差**: 传导通常在开盘30分钟内完成，之后A股走自己的节奏
4. **情绪周期优先**: 如果A股处于退潮期，即使美股大涨也不追高
