# 游资选股方法论 (Stock Selection Methodology)

## 7步选股流程

### Step 1: 情绪周期检查
用 `detect_sentiment_phase` 获取当前阶段。
- 冰点期(freezing): 不买。等待。保持现金。
- 退潮期(ebb): 不买。现金为王。
- 启动期(ignition): 小仓位试探，只做龙头首板。
- 加速期(acceleration): 积极参与，跟随主线，可加仓。
- 高潮期(climax): 谨慎，只做确定性最高的龙头。

### Step 2: 扫描涨停池
用 `get_opportunity_candidates` 获取市场扫描器发现的候选标的。
关注: 首板/二板候选，非ST，非次新（上市<60天）。
评分 > 60 = 重点关注，评分 > 70 = 龙头确认。

### Step 3: 连板晋级率
用 `get_consecutive_board_rate` 检查市场健康度。
- 晋级率 > 25% = 情绪健康，积极参与
- 晋级率 15-25% = 一般，精选个股
- 晋级率 < 15% = 情绪衰退，谨慎
- 晋级率 < 10% = 退潮信号，不参与新仓

### Step 4: 龙头评分
扫描结果中包含 LeaderDetector 评分（满分100）。
- > 70分 = 龙头确认，重点关注
- 60-70分 = 候选龙头，需深入分析
- 50-60分 = 一般标的，不优先
- < 50分 = 不参与

评分维度：首封时间(30分) + 封单强度(20分) + 板块跟风(15分) + 资金共识(20分) + 开板回封(15分)

### Step 5: 板块联动验证
同板块涨停数 >= 3只 = 板块共识确认（主线）
同板块涨停数 = 1只 = 独板涨停，风险较高
跨板块验证: 用 `get_concept_heat` 查看概念板块热度

### Step 6: 资金流验证
用 `get_intraday_fund_flow_timeline` 检查主力资金方向。
- 主力净流入 + 涨停 = 健康，可参与
- 主力净流出 + 涨停 = 危险信号（可能诱多/出货）

### Step 7: 入场/出场参数
- 买入时间: 14:30-15:00（尾盘买入，减少日内波动风险）
- 卖出时间: 09:30-10:00（次日早盘，高开兑现）
- 止损: 成本价下方3-5%
- 目标: 涨停板(+10%)或连板接力
- 仓位: 根据情绪周期（Step 1）确定上限
  - 启动期: 单票 ≤ 20%，总仓位 ≤ 50%
  - 加速期: 单票 ≤ 25%，总仓位 ≤ 80%
  - 高潮期: 单票 ≤ 15%，总仓位 ≤ 60%

## 重要原则

1. **不追涨**: 已经大涨5%+的不追，等回调或尾盘
2. **不抄底**: 下跌趋势不接，等止跌信号
3. **龙头优先**: 板块涨停多只时，只做龙头（评分最高）
4. **分歧买入**: 最佳买入点是市场有分歧时（换手率15-25%）
5. **一致卖出**: 市场一致看好时（量缩板）是考虑卖出的时候
