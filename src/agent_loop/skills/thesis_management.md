# 投资论文管理 (Thesis Management)

## 每个持仓必须有论文

买入前必须回答：
1. **为什么买？** — 明确的逻辑链（事件→影响→机会）
2. **什么价位买？** — 具体的入场条件
3. **什么时候卖？** — 目标价 + 最大持有天数
4. **什么时候止损？** — 具体的止损条件（价格或事件）
5. **论文什么时候失效？** — 如果X天内逻辑未兑现，退出

## 论文生命周期

```
创建(active) → 增强(confidence↑) → 弱化(weakening) → 失效(invalidated)
                                   ↗ 实现(realized)
```

### 置信度衰减
- 默认每天衰减2%（0.02/天）
- 如果有新的支持证据，置信度回升
- 置信度 < 35% → 弱化状态（考虑减仓）
- 置信度 < 20% → 自动失效（触发卖出）

### 论文失效条件
- 时间到期（默认5个交易日）
- 价格击穿止损位
- 买入逻辑发生根本变化（如政策反转）
- 置信度自然衰减到阈值以下

## 使用工具管理论文

- `get_active_theses` — 查看所有活跃论文
- `create_thesis` — 买入时创建论文
- `update_thesis` — 有新证据时更新

## 关键规则

1. **无论文不买入** — 每笔交易必须有明确的论文
2. **论文失效必须卖出** — 不要"再等等"
3. **一个股票一个论文** — 不要叠加多个理由
4. **论文要可证伪** — 必须有明确的失效条件
