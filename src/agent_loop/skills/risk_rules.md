# 风险管理规则

## 仓位控制

- **单票上限**: 25%总资金
- **板块集中度上限**: 40%
- **总仓位**: 根据情绪周期动态调整（见sentiment_framework）

## Kelly公式定仓

```
f = (b*p - q) / b
f: 建议仓位比例
b: 盈亏比 = 目标收益 / 止损距离
p: 胜率（来自贝叶斯后验）
q: 1-p
实际仓位 = f × 0.5 (半Kelly，更保守)
```

## 止损规则

1. **ATR止损**: 默认2倍ATR（14日）
2. **论文失效止损**: 如果买入逻辑不再成立，无条件卖出
3. **日度风险预算**: 单日最大亏损3%总资金
4. **连续亏损**: 连续3次止损 → 仓位减半一天
5. **峰值回撤**: 从峰值回撤10% → 全部仓位减半

## 隔夜风险 (T+1)

- 今天买入的股票明天才能卖
- 买入前必须评估隔夜持有的风险
- 利空出尽的股票隔夜风险较低
- 高位追涨的股票隔夜风险最高

## 熔断条件

| 条件 | 动作 |
|------|------|
| 组合3日亏损>5% | 暂停所有买入 |
| 连续3次止损 | 休息1个交易日 |
| 峰值回撤>10% | 所有仓位减半 |
| 系统置信度<40% | 只允许现金 |
