# 情绪周期框架 (Sentiment Cycle Framework)

A股超短线交易的核心元框架。来源：炒股养家、龙飞虎等顶级游资方法论。

## 5个阶段

### 1. 冰点期 (Freezing) — 极度保守
- 特征：跌停多、涨停少、连板高度<3、北向资金净流出、量能萎缩
- 操作：仓位≤20%，单票≤5%，止损1%
- 策略：只观察不操作，等待反转信号（恐慌见底、超跌反弹龙头）
- 买什么：不买。如果非要操作，只做超跌反弹龙头第一板

### 2. 启动期 (Ignition) — 开始试探
- 特征：跌停减少、出现2-3个涨停、龙头开始连板、情绪修复
- 操作：仓位≤50%，单票≤10%，止损1.5%
- 策略：小仓位试探龙头股，确认则加仓
- 买什么：新周期的第一个龙头首板

### 3. 加速期 (Acceleration) — 积极操作
- 特征：涨停数量增多、龙头连板高度上升、板块扩散、量能放大、北向资金净流入
- 操作：仓位≤80%，单票≤15%，止损2%
- 策略：跟随龙头和板块主线，敢于加仓
- 买什么：主线板块的龙头股、补涨龙、高位龙头分歧日

### 4. 高潮期 (Climax) — 开始减仓
- 特征：涨停爆量（但质量下降）、垃圾股乱涨、高位股开始分化、一致看多
- 操作：仓位≤50%，单票≤10%，止损1.5%
- 策略：逐步减仓，只保留最强龙头
- 卖什么：非龙头的跟风股、走弱的板块

### 5. 退潮期 (Ebb) — 全面防守
- 特征：龙头断板、跌停潮、恐慌蔓延、量能骤降
- 操作：仓位≤10%，单票≤3%，止损1%
- 策略：清仓观望，现金为王
- 卖什么：全部卖出。不要抄底。

## 关键规则

1. **情绪决定仓位**：不是个股好不好，是市场情绪允不允许
2. **做多只在启动和加速期**：其他阶段以防守为主
3. **龙头才值得持有**：非龙头在退潮期最先被砸
4. **现金是最好的武器**：在冰点和退潮期，现金就是alpha
