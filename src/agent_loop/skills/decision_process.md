# 决策流程

## 买入决策5步法

### Step 1: 基础概率
这个板块/类型的股票，历史胜率是多少？
用 `get_outcome_stats` 查看历史信号准确率。

### Step 2: 新证据
有什么新信息改变了基础概率？
- 政策面：利好/利空？
- 资金面：主力流入/流出？用 `get_fund_flow` 查看
- 技术面：突破/支撑？用工具查看走势
- 情绪面：市场处于什么阶段？用 `detect_sentiment_phase` 查看

对每条证据判断：看多/看空/中性，强度1-5。

### Step 3: 组合适配
当前组合能容纳这笔交易吗？
- 同行业集中度是否过高？
- 总仓位是否超出情绪阶段限制？
- 风险预算还剩多少？

### Step 4: 风险场景
- 最好情况（概率X%）：...
- 基本情况（概率Y%）：...
- 最坏情况（概率Z%）：...

### Step 5: 最终决策
综合以上分析：
- 行动：买入/观望/放弃
- 价位区间
- 仓位大小（Kelly公式建议）
- 止损位
- 目标价
- 持有天数
- 失效条件

## 收敛原则

**至少2个独立信号域同意才能买入：**
- 技术面 + 资金面 ✅
- 事件面 + 龙头检测 ✅
- 技术面 + 日内模式 ❌ （同源信号，不独立）

## 辩论原则

每笔买入决策前，必须同时考虑：
- **看多论点**: 为什么应该买？
- **看空论点**: 为什么不应该买？
- **裁决**: 哪边证据更强？为什么？
