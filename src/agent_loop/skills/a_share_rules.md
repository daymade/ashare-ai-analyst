# A股交易规则

## 基本规则

- **T+1**: 今天买入的股票，明天才能卖出。今天持有的才能今天卖。
- **最小单位**: 100股（1手）
- **涨跌停**: 主板±10%，创业板(300xxx)/科创板(688xxx)±20%
- **ST股**: ±5%涨跌停

## 交易时间 (CST)

| 时段 | 时间 | 说明 |
|------|------|------|
| 集合竞价 | 09:15-09:25 | 可报价不可撤单(09:20-09:25)，确定开盘价 |
| 早盘 | 09:30-11:30 | 连续竞价 |
| 午休 | 11:30-13:00 | 停止交易 |
| 午盘 | 13:00-15:00 | 连续竞价 |
| 收盘竞价 | 14:57-15:00 | 确定收盘价 |

## 最佳操作时段

- **卖出窗口**: 09:30-10:00 — 利用隔夜情绪溢价卖出
- **买入窗口**: 14:30-15:00 — 确认全天走势后买入，减少隔夜风险
- **避免操作**: 11:00-11:30（午盘前砸盘多）、14:00-14:30（尾盘前不确定性大）

## 集合竞价要点

- 09:15-09:20: 可自由撤单，虚假报价多
- 09:20-09:25: 不可撤单，真实意图暴露
- 高开>3%: 关注是否有利好消息支撑
- 低开>3%: 评估是否需要开盘卖出

## 涨跌停板规则

- 涨停 = 买盘远大于卖盘，价格被锁定
- 封板 = 涨停后大量买单排队（封单）
- 炸板 = 涨停后封单被吃完，价格回落
- 封板强度 = 封单金额 / 当日成交额
