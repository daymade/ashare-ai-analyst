# 龙头股识别 (Leader Stock Detection)

## 6维评分体系

### 1. 先手优势 (First Mover) — 权重25%
- 板块内第一个涨停的股票
- 首板比二板更有价值（确认市场认可）
- 连续先手 > 单次先手

### 2. 封板强度 (Seal Strength) — 权重20%
- 封单量/成交量 比值越高越强
- 封板时间越早越好（10点前封板最佳）
- 开板后快速回封 = 强势信号

### 3. 板块跟随度 (Sector Followers) — 权重15%
- 龙头涨停后，同板块有多少跟风涨停
- 跟风越多 = 板块共识越强
- 独板（无跟风）风险较高

### 4. 资金共识 (Capital Consensus) — 权重15%
- 龙虎榜机构+游资同时买入 = 强共识
- 只有游资买 = 短线投机
- 机构大量卖出 = 危险信号

### 5. 连板韧性 (Board Resilience) — 权重15%
- 连板过程中是否有分歧转一致
- 分歧日（开盘低开/盘中开板）后回封 = 确认龙头
- 一字板连续涨停无参考价值（没有换手）

### 6. 微观结构 (Microstructure) — 权重10%
- 封单队列深度
- 大单净流入占比
- 分时图量价配合

## 龙头确认条件

满足以下任一：
- 6维总分 ≥ 70分
- 连板高度 ≥ 5板
- 板块内连续3天排名第一

## 操作规则

- **买入时机**: 分歧日（开盘弱势→午后回封）
- **卖出时机**: 一致转分歧（全天高开→尾盘炸板）
- **不追涨**: 一字板不追，等开板换手
